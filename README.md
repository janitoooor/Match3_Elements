Юнити проект находится в папке match3_elements_unity

Последний собранный апк билд можно взять из папки apk_builds

Сцена для запуска игры в проекте Юнити - Launcher

Чтобы двигать блоки в редакторе (считывался тач), нужно запускать в simulator моде

Чтобы очистить сейв в редакторе - Tools/Clear Player Prefs Saves

Все настройки игры вынесены в конфиги в папке Configs (кроме одной настройки - скорости передвижения блоков, это значение задается в префабе Block)

Все префабы вынесены в папку Prefabs

В проекте используется зенжект, поэтому отдельно сцены не все получится запустить. В целом проект поделен на три части - core, meta и common/base
